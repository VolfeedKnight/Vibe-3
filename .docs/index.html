PRD 요약

팀원 스케쥴 관리

휴가, 근무, 출장 일정을 공유하는 캘린더를 제공한다. 일정 CRUD, 유형별 색상, 팀원/유형 필터를 핵심으로 한다.

엑셀 업무 자동화

엑셀 파일을 특정 컬럼 기준으로 분리하거나 여러 파일을 병합한다. 컬럼 검증과 결과 다운로드를 지원한다.

민원 대응 챗봇

민원 매뉴얼을 기반으로 대응 방향과 답변 초안을 생성한다. 근거 문서 표시와 사용자 최종 검토를 기본 원칙으로 한다.

뉴스 기사 수집

공공 행정 관련 뉴스를 매일 아침 수집하고, 제목, 언론사, 발행일시, 요약, 원문 링크를 제공한다.

MVP 범위

  • 팀원 일정 등록, 조회, 수정, 삭제
  • 엑셀 파일 컬럼 기준 분리와 병합
  • 민원 매뉴얼 업로드와 답변 초안 생성 화면
  • 공공 행정 뉴스 수동 수집과 목록 표시

아키텍처 요약

영역 기술 역할
Frontend TypeScript, Vite, React 사용자 화면, 파일 업로드, 캘린더, 챗봇 UI, 뉴스 목록

운영 요약

현재 프로젝트는 구현 전 준비 단계입니다. FE는 모듈 설치까지 완료됐고, BE는 uv 가상환경과 FastAPI 설치까지 완료됐습니다.

환경 확인

node --version
npm --version
uv --version
git --version

Backend Python 실행

cd backend
$env:UV_CACHE_DIR='..\.uv-cache'
uv run python --version

예상 실행 명령

cd frontend
npm run dev
cd backend
$env:UV_CACHE_DIR='..\.uv-cache'
uv run fastapi dev app/main.py
